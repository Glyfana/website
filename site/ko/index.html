SHA256 확인

다운로드 파일의 해시를 운영체제별로 비교하세요.

WindowsSHA256

SHA256 2409822E41301128F8E586BB116006E9C7599B884C1D7636191854754651CDF7

PowerShell Get-FileHash ".\Glyfana-0.1.9.Setup.exe" -Algorithm SHA256

현재 공개 설치 파일은 코드 서명이 없어 첫 실행 시 Windows SmartScreen 경고가 나타날 수 있습니다. 안내가 뜨면 추가 정보 → 실행을 선택하세요.

Mac OSSHA256

SHA256 381AA0E27046C5A79C985452EA23F38099D1C83E8EC3675EE3A90776FADBBE90

Terminal shasum -a 256 ./Glyfana-0.1.9-x64.dmg

현재 빌드는 코드 서명과 노터라이즈가 없어 첫 실행 시 Gatekeeper 안내가 나타날 수 있습니다. 시스템 설정에서 열기를 허용하세요.

LinuxSHA256

SHA256 4BE5F3D892C10EE95E2CD16ED32591D62F63E27BCA32D75C2C83FFDA38E545AE

Terminal sha256sum ./Glyfana-0.1.9-x86_64.AppImage

AppImage는 실행 권한을 준 뒤 바로 실행할 수 있고, .deb는 Ubuntu 계열에서 설치할 수 있습니다.

FAQ

자주 묻는 질문

무료로 받을 수 있나요?

네. GitHub Releases에서 최신 공개 빌드를 받을 수 있습니다.

네. 릴리스 노트와 SHA256 값을 함께 제공합니다.
