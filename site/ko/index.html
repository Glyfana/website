FAQ

자주 묻는 질문

클라우드 계정이 필요한가요?

아니요. 로컬 Markdown 파일을 바로 열고 저장할 수 있습니다.

기존 Markdown 파일도 열 수 있나요?

네. 기존 .md 파일을 바로 열 수 있습니다.

현재 지원하는 플랫폼은 무엇인가요?

현재 공개 설치 파일은 Windows용입니다.
