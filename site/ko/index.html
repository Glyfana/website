최신 변경점

다운로드 페이지 안에서 최신 릴리스 핵심 내용을 바로 확인할 수 있습니다.

GitHub에 릴리스 노트가 게시되면 이 영역에서 중요한 줄을 먼저 보여주고, 전체 노트는 한 번에 열 수 있게 연결합니다.

집중형 편집

Glyfana를 Typora 스타일의 Markdown 에디터로 소개하고, 문서에 몰입할 수 있는 흐름을 전면에 둡니다.

복구 기본 탑재

자동 저장, 초안 복구, 충돌 처리 같은 핵심 안정성 기능을 설치 버튼 바로 옆에서 설명합니다.

릴리스 전달

설치 파일, 릴리스 노트, 자산 목록을 GitHub와 연결해 사용자가 설치 전에 근거를 직접 확인하게 합니다.

설치

추가 배포 인프라 없이도 Windows 설치 파일을 배포할 수 있습니다.

GitHub Releases는 그대로 아티팩트 저장소로 두고, 이 페이지는 다운로드, 설치, 검증 안내를 한 번에 제공하는 더 나은 진입점 역할을 합니다.

01

현재 설치 파일 다운로드

기본 버튼으로 현재 안정 릴리스에 연결된 .exe 설치 파일을 바로 받을 수 있습니다.

02

파일 해시 검증

설치 전에 로컬 SHA256 값과 공개된 체크섬을 비교해 파일 무결성을 확인합니다.

03

설치 후 변경 사항 확인

릴리스 노트와 전체 자산 목록이 연결되어 있어 배포 내용 전체를 빠르게 검토할 수 있습니다.

검증

설치 전에 신뢰 근거를 먼저 보여줍니다.

체크섬, PowerShell 검증 명령, 공개된 릴리스 자산을 설치 시작 전에 한 자리에서 확인할 수 있게 합니다.

무결성

PowerShell로 설치 파일 해시를 계산해 공개된 SHA256 값과 비교하세요.

SHA256 대기 중

PowerShell Get-FileHash ".\installer.exe" -Algorithm SHA256
