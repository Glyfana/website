핵심 기능

처음 써도 바로 이해되는 세 가지에 집중했습니다.

복잡한 구현 설명보다, 실제로 자주 쓰는 작업 흐름만 먼저 보여줍니다.

작성 경험

읽기 좋은 화면에서 바로 쓸 수 있습니다.

문서 흐름을 보면서 바로 쓰고 다듬을 수 있어 긴 글도 끊기지 않고 이어집니다.

소스 모드

필요할 때만 Markdown 원본으로 전환합니다.

구조를 직접 만져야 할 때는 소스 모드로 바로 전환해 세밀하게 수정할 수 있습니다.

로컬 파일

노트와 이미지를 파일 기준으로 안전하게 다룹니다.

자동 저장과 변경 감지로 작업 손실을 줄이고, 로컬 파일 흐름을 단순하게 유지합니다.

SHA256 대기 중

PowerShell Get-FileHash ".\installer.exe" -Algorithm SHA256

현재 공개 설치 파일은 코드 서명이 없어 첫 실행 시 Windows SmartScreen 경고가 나타날 수 있습니다. 안내가 뜨면 추가 정보 → 실행을 선택하세요.

FAQ

자주 묻는 질문

다운로드 전에 많이 묻는 내용만 짧게 정리했습니다.

클라우드 계정이 필요한가요?

아니요. 로컬 Markdown 파일을 바로 열고 저장할 수 있습니다.

기존 Markdown 파일도 열 수 있나요?

네. 이미 쓰고 있는 .md 파일을 열어 그대로 이어서 작업할 수 있습니다.

현재 지원하는 플랫폼은 무엇인가요?

이 페이지에서 제공하는 공개 설치 파일은 현재 Windows용입니다.
